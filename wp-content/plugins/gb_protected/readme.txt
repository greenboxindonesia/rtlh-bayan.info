== Description ==

Password Protect Wordpress plugin for Wordpress is an easy to use plugin to restrict all access to your blog by the use of a single password. Examples of use:

Family blog.

Blog in development for a client.

Follow @spidersdesign

== Installation ==

This section describes how to install the plugin and get it working.

 

1. Unzip archive

2. Upload \`password-protect-wordpress\` to the \`/wp-content/plugins/\` directory

3. Activate the plugin through the 'Plugins' menu in WordPress

1. Edit the configuration by clicking on 'password protect' under settings
== Frequently Asked Questions ==
